使用（Usage)

目录由命令行工具 gitbook-summary 自动生成。自由写作、发布，搭建自出版平台的方法，请点击这里

简要介绍如下：

(1)克隆源文

$ git clone https://github.com/jxzsxsp/java-concurrent-programming.git

(2)安装gitbook

$ npm install -g gitbook-cli

(3)安装依赖包

cd java-concurrent-programming
npm install
gitbook install

(4)写作构建

写作，并开启服务（构建）

$ gitbook serve

通过http://localhost:4000实时浏览

(5)生成目录

只要修改了文章标题和文件夹，就应该重新生成目录文件

$ npm run summary

(6)一键发布

$ npm run deploy

以后，只要4-6的过程就是了。
